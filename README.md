# Crypto Dashboard Frontend

Next.js 14 dashboard UI for crypto monitoring data.

## Features
- Dark theme dashboard
- Summary stats cards
- Transaction list
- Alert history with score badges
- AI analysis page
- API integration using `NEXT_PUBLIC_API_URL` and `NEXT_PUBLIC_API_KEY`

## Local run
```bash
cp .env.example .env
npm install
npm run dev
```

Default app URL: `http://localhost:3000`
